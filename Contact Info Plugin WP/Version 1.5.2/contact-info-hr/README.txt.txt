== English ==
This plugin adds contact options in the admin and automatically updates Elementor widgets (icon, title, and description) with phone and email links.

== Description ==
- Allows you to configure Phone and Email from the admin.
- Updates icon, title, and description in header, footer, and contact page.
- Compatible with Elementor widgets using classes: .cntemail, .cntnum, .hmail, .hcallus, .foomail, .foonum.


== Spanish ==
Este plugin agrega opciones de contacto en el admin y actualiza automáticamente los widgets de Elementor (ícono, título y descripción) con los links de teléfono y correo electrónico.

== Description ==
- Permite configurar Teléfono y Email desde el admin.
- Actualiza ícono, título y descripción en header, footer y página de contacto.
- Compatible con widgets de Elementor usando clases: .cntemail, .cntnum, .hmail, .hcallus, .foomail, .foonum.
